## About
I am a cybersecurity professional and Penn State graduate student focused on practical SOC engineering.

I enjoy building detections that reduce analyst fatigue, improving triage quality, and automating repetitive security workflows.

Little personal rule: if a playbook is not usable at 2 AM, it is not finished.

## Experience Snapshot
- Security Operations Engineer experience at Workato: alert triage, endpoint investigations, detection tuning, and policy enforcement.
- Security lab engineering at Trainocate: built hands-on virtual labs and analyzed IDS/network telemetry.
- MPS in Cybersecurity Analytics and Operations, Penn State (GPA 3.89).

## Topics People Usually Ping Me About
- Reducing false positives without losing real coverage
- Building SOC playbooks analysts actually follow
- Turning Python scripts into repeatable security workflows
- Blue-team lab design for practical training
